# Clearview (JSX)

A lean Project Management web app (day tiles + Gantt) in plain **React JSX** with **Vite** and **Tailwind**.

## Dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Notes
- Uses `framer-motion` and `lucide-react`.
- Data persists in `localStorage` for demo purposes.
